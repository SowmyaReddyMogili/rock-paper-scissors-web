# ✊✋✌️ Rock Paper Scissors (Web)

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript where the user plays against the computer.

## 🚀 Features
- Play against computer
- Random computer choices
- Live score tracking
- Interactive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 How to Run
1. Clone or download the repository
2. Open index.html in your browser
